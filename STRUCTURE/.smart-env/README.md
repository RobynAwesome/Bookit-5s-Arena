# Smart Env

This folder mirrors the Schematics vault shape but only keeps project-relevant placeholders.

Subfolders:
- `embedding_models`: reserved for local embedding metadata if the Obsidian setup adds semantic indexing later.
- `event_logs`: reserved for exported implementation and audit events.
- `multi`: reserved for generated multi-note context bundles.
- `smart_components`: reserved for reusable structured note fragments.
- `smart_contexts`: reserved for linked context packs for future maintainers.
