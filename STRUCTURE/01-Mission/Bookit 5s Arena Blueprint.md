# Bookit 5s Arena Blueprint

## Mission

Bookit 5s Arena is both a venue website and an operating system for a 5-a-side football business:

- attract guests
- convert players into registered users
- manage courts and bookings
- run leagues and tournaments
- give managers match-day tooling
- give admins live control over content, fixtures, rights, newsletters, integrations, and security surfaces

## Product Pillars

- Venue commerce: courts, events, bookings, rewards, referrals
- Competition engine: leagues, tournament brackets, standings, fixtures, stats
- Content and trust: blog, docs, case studies, partners, roadmap, security
- Operational tooling: admin dashboards, rights, analytics, newsletters, integrations, sandbox

## Current North Star

The biggest shipped change in this implementation cycle is the Premier League-first fixtures hub plus the admin tooling around integrations and security. The site is no longer just a venue frontend; it now has:

- a live football data experience
- media enrichment via YouTube and search fallbacks
- integrations observability
- admin sandbox workflows
- hardened auth and anti-bot layers
